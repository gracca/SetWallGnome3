Set the wallpaper in Gnome 3
============================

A simple program to set the wallpaper in Gnome 3.

It is written in [Python](http://www.python.org/) and [Gtk+ 3](http://www.gtk.org/).

How to run
----------

Make it executable:

    $ chmod +x SetWallGnome3.py

and then run it:

    $ ./SetWallGnome3.py
